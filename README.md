# Solid Glass
![preview](https://github.com/user-attachments/assets/a8eca5f9-deeb-4516-b1ca-ca0f18b94943)


Apple calls their new design language Liquid Glass so naturally I had to call this Solid Glass. Get it?

Solid glass is a theme that brings the fresh iOS 26 Liquid Glass icons to older jailbreakable devices, now with Dark Mode and 3rd party app icon support! Optimized to feel native on older versions, from icon corner radius to contrast levels. 

![branding](https://github.com/user-attachments/assets/91c3476f-ed3d-4a38-af7d-f2634d43bae3)

# Compatibility
Solid Glass supports any jailbreakable iOS version - both rootful and rootless,
apply using Anemone/SnowBoard and enjoy!(Winterboard support coming soon)
