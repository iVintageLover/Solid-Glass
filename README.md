# Solid Glass

Apple calls their new design language Liquid Glass, but when you shrink them down to PNGs, it kind of becomes Solid. Get it?

# Introducing Solid Glass 27
Featuring brand new iOS 27 icons and a brand new Clear Light variant
built around Human Interface Guidelines

<img width="1104" height="767" alt="preview new" src="https://github.com/user-attachments/assets/eec8f816-0f66-44cc-8e7b-1e032d82b221" />


### 
Solid Glass is a theme that brings the fresh new iOS 27 Liquid Glass icons to older jailbreakable devices. Now with 3 beautiful modes: Light, Clear Light, Clear Dark and regular Dark. Optimized to feel native on older versions, from icon corner radius to contrast levels. 

![branding](https://github.com/user-attachments/assets/91c3476f-ed3d-4a38-af7d-f2634d43bae3)

# Compatibility
Solid Glass supports any iOS version on both rootful and rootless jailbreaks,
apply using Anemone/SnowBoard and enjoy!(Winterboard support coming soon)
